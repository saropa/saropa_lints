## Issues & Inline Annotations

**Issues tree** — Violations grouped by severity (Error, Warning, Info) then by file. Click to jump to the source.

**Inline annotations** — Error Lens-style decorations show violation messages at the end of each line, colored by severity. Toggle via settings.

**Filters** — Filter by text, severity, impact, or rule name. Right-click to suppress noisy folders, files, or rules.

**Code Lens** — A lens at the top of each Dart file shows issue count. Click to focus the Issues view.

**Quick fixes** — Right-click a violation to apply a fix without opening the file. "Fix all in this file" for bulk fixes.
