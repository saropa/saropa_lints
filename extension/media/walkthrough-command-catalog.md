## Browse All Commands

The extension has 115+ commands across analysis, security, package health, reporting, and more.

The **Command Catalog** is a searchable, categorized view of every command:

- **Search** — type to instantly filter by title or description
- **Categories** — collapsible sections for each feature area
- **Click to execute** — run any command directly from the catalog
- **Context-menu commands** — toggle visibility of commands normally available only via right-click

Open the catalog from the command palette: **Saropa Lints: Browse All Commands**.
